# 项目功能列表 (FEATURE)

本文件详细记录了 GoldBar 已实现的所有功能点，以便在更换 AI 协作或重新构建项目时作为核心参考。

## 💎 核心功能

### 1. 实时金价追踪 (Real-time Gold Price)
- **数据源**: `https://api.gold-api.com/price/XAU`
- **内容**: 追踪 XAU (伦敦金) 的国际实时行情（美元/盎司）。
- **刷新频率**: 支持 15 秒 / 30 秒 / 1 分钟 / 2 分钟四档自定义刷新。

### 2. 货币与单位换算 (Currency & Unit Conversion)
- **多币种支持**: 支持 **人民币 (CNY)**、**美元 (USD)**、**日元 (JPY)**、**欧元 (EUR)**。
- **汇率多源容错**: 主接口失败时自动回退到备用接口。
- **汇率数据源**:
  - `https://open.er-api.com/v6/latest/USD`
  - `https://api.exchangerate-api.com/v4/latest/USD`
  - `https://api.frankfurter.app/latest?from=USD`
- **逻辑**: 自动获取对应币种的实时汇率，并显示当前汇率来源。
- **换算公式**: `(国际金价 * 目标币种汇率) / 31.1034768`
- **目标单位**: 对应币种/克。
- **持久化**: 应用会记住用户选择的币种偏好。

### 6. 币种切换 (Currency Selection)
- **快捷切换**: 详情面板中增加分段选择器 (Segmented Picker)，支持一键切换计价币种。
- **动态 UI**: 菜单栏和详情面板会根据选定币种动态更新符号（如 ¥, $, €）和数值。

### 3. macOS 菜单栏集成 (Menu Bar Extra)
- **常驻显示**: 在 macOS 菜单栏直接显示单价。
- **状态图标**: 使用自定义金色徽章图标（GoldBadgeIcon）+ 动态涨跌箭头 (▲/▼)。
- **无 Dock 干扰**: 应用以 `.accessory` 策略运行，不会在 Dock 栏显示。

### 4. 交互式详情视图 (Detail Popover)
- **实时看板**: 点击菜单栏图标弹出。
- **辅助信息**: 显示当前汇率、汇率来源、国际原始金价（美元/盎司）。
- **更新时间**: 记录上一次数据获取的时间。
- **涨跌幅统计**: 基于 USD 原始价格计算百分比，切换币种不会改变涨跌幅逻辑结果。
- **历史图表**: 内置历史金价折线图（最近最多 120 个点，按当前币种/克展示）。

### 5. 操作控制 (Controls)
- **立即刷新**: 强制触发 API 调用获取最新数据。
- **退出应用**: 正常关闭程序。

### 7. 刷新间隔控制 (Refresh Interval Control)
- **用户可选**: 提供分段选择器，支持 15 秒、30 秒、1 分钟、2 分钟。
- **即时生效**: 调整后立即重建定时器并生效。
- **持久化**: 重启应用后自动恢复上次选择。

## 🏗️ 技术实现方案
- **框架**: SwiftUI (macOS 13+).
- **架构**: MVVM (GoldPriceManager 作为 ObservableObject 控制数据流)。
- **并发处理**: 使用 Swift Concurrency (async/await) 进行异步 API 调用。
- **状态监控**: 使用 Combine 定时触发刷新任务。
