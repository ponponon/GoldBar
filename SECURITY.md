# SECURITY.md - 安全政策

本文件描述了 GoldBar 项目的安全政策。

## 🔒 漏洞报告
如果你发现了本项目中的安全漏洞，请通过 GitHub Issues 进行反馈。我们将尽快评估并处理。

## 🔑 API 安全
- 本项目目前使用 **公开免费 API** 获取数据。
- 建议用户在 fork 或二次开发时，如果需要更高频次的刷新，申请自己的 API Key 并妥善保管。
- 禁止将包含个人 API Key 的代码推送到公共分支。

## 📦 依赖管理
- 本项目依赖 Swift Package Manager (SPM) 进行依赖管理。
- 核心逻辑尽量保持零第三方依赖，以最大化安全性。

## ⚠️ 免责声明
- GoldBar 提供的价格仅供参考，不作为投资建议。
- 数据的准确性取决于上游 API 供应商的质量。
