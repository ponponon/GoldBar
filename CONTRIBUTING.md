# 贡献指南 (CONTRIBUTING)

感谢你对 GoldBar 项目的关注！本指南将帮助你了解如何参与开发。

## 🛠️ 环境准备

1.  **macOS**: 本项目是一个原生 macOS 应用。
2.  **Xcode 14+**: 建议使用最新版本的 Xcode。
3.  **Swift 5.7+**: 核心逻辑和 UI 均使用 Swift 构建。

## 🏗️ 提交代码

1.  **Fork** 本项目。
2.  **创建功能分支**: `git checkout -b feature/your-feature-name`。
3.  **提交更改**: `git commit -m "Add some feature"`。
4.  **推送到分支**: `git push origin feature/your-feature-name`。
5.  **开启 Pull Request**。

## 📝 提交规范

- 请确保代码风格与现有代码一致。
- 提交信息请使用清晰、简明的中文或英文描述。
- 如果引入了新的依赖，请在 PR 中说明原因。

## 🐞 报告问题

如果你发现了 Bug 或有新的功能建议，请通过 GitHub Issues 提交。
提交时请附带以下信息：
- 你的 macOS 版本。
- 复现步骤（如果是 Bug）。
- 期望的行为。
